# Brand Guide and Website Instructions

This directory contains the brand guide and instructions for the Mythic Rx website.

## Contents

### Brand Assets
- **Mythic - rx- white.ai** - White version of the Mythic Rx logo (Adobe Illustrator file)
- **Mythic Green logo.ai** - Green version of the Mythic logo (Adobe Illustrator file)
- **mythic-rx M.ai** - Mythic Rx "M" logo mark (Adobe Illustrator file)
- **Mythic Brand assets (1).ai** - Complete brand asset package (Adobe Illustrator file)

### Documentation
- **Mythic Brand Guide.lnk** - Link to the complete brand guide
- **Mythic website Questionnaire.docx** - Website questionnaire with requirements and preferences
- **MythicRx SOW Final Draft.docx** - Statement of Work (SOW) document outlining project scope

## Usage

### Working with Brand Assets
The `.ai` files contain vector graphics for the Mythic Rx brand. These files require Adobe Illustrator to open and edit. Key brand assets include:

1. Logo variations (white, green, and M mark)
2. Complete brand asset package with all elements

### Understanding Requirements
- Review the **Questionnaire** document to understand website goals, target audience, and feature requirements
- Reference the **SOW** document for project scope and deliverables

## Brand Consistency

When developing the website, ensure:
- Use official logo files from this directory
- Follow brand guidelines for colors, fonts, and visual style
- Maintain consistency across all pages
- Reference the brand guide for approved usage

## File Formats

- `.ai` files - Adobe Illustrator vector graphics (editable)
- `.docx` files - Microsoft Word documents
- `.lnk` files - Windows shortcuts (may require adjustment for other platforms)

## Questions?

If you need additional information about the brand assets or website requirements, please refer to the documents in this directory or contact the project stakeholders.
